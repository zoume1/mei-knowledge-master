# 什么是Electron

这是一个跨平台的桌面应用开发框架,它的本质是Chromim 与node.js+NativeAPI

开发文档 https://www.electronjs.org/zh/docs/latest/api/base-window

# Electron流程模型

Main（主进程） node环境

作用  可以和渲染进程通讯， 可以操作文件，可以调用native API

Render （渲染进程） 页面实现  

# Electron模块生命周期

## 1.app事件

### 事件：will-finish-launching

当应用程序完成基础的启动的时候被触发

### 事件: ready

当 Electron 完成初始化时，触发一次

### 事件: 'window-all-closed'

当所有的窗口都被关闭时触发。

### 事件：'before-quit'

在程序关闭窗口前发信号。 调用 `event.preventDefault()` 将阻止终止应用程序的默认行为。

### 'will-quit'

当所有窗口被关闭后触发，同时应用程序将退出。 调用 `event.preventDefault()` 将阻止终止应用程序的默认行为。
