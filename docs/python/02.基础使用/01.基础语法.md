```
print(10) # 输出整数10
print(13.14) # 输出浮点数13.14
print("我是最棒的") 
```

## ****变量和标识符

### 变量

a=666

创建了一个变量a，并且把666 赋值给了a

**注意后续重新赋值a  不会重新分配内存，直接修改原来的值 变量可以重复赋值，并且可以是不同的数据类型**

### 标识符

1. 只能由数字，字母，下划线组成
2. 不能数字开头
3. 不能是关键字
4. 严格区分大小写

## 数值类型

1. 整数  int
2. 浮点型 float
3. 布尔类型  bool True False
4. 复数 complex

检测数字类型  type（变量名）

## 格式化输出（生成一定格式的字符串）

%   %s 占位符    %d 数字  %.1f 浮点数 （默认后六位小数，.1代表保留一位，超过四舍五入） %% 输出一个%

```
name = 'mei'
age = 17
print("我的名字: %s,我的年龄: %04d"%(name,age))
```

f格式化

```
print(f"我的名字{name}")
```
