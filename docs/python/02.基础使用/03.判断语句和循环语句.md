## ##if语句的基本格式

语法：
if 要判断的条件

```
name = 'mei'
age = 10.1
if age > 10:
    print('hahahah')
    print('lalalal')

else :print('lalalal')
```

## 循环语句

格式  for 临时变量 in 序列类型

```
t1 = (1,2,3,4,5,6)

for element in t1:
    print(f"元组的元素有：{element}")
```
