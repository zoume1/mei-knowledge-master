## 字符串的操作

### +  字符串拼接

```
print('aa'+'bb')  //输出aabb 字符串拼接
```

### 字符串运算符

重复输出

```
print('哈哈哈\n'*4)  输出4遍 哈哈哈
```

成员运算符

in：如果包含的话返回True，不包含返回False

not in ： 和上面相反

```
print('m' in name)  输出 True
```

### 下标

写法 字符串[下标]  从0开始

注意 从右到左  -1 -2

### 截取

语法 [开始位置：结束位置：步长]  注意包前不包后的原则，不包含结束位置   步长的绝对值大小表示截取的间隔，正负代表方向

```
print(name[0:2]) 
print(name[1:]) // 下标1以后的全部截取到
print(name[:2]) // 下标2以前的全部截取到 不包含2
name = 'skdjgfklsdjglk;jsdljg;lsdjkgl;'
print(name[0:8:2])  //输出 sdgk
```

### 查找

find

```
print(name.find('m'))  //返回下标
```

### 返回出现次数

count

```
print(name.count('m'))  
```

### 替换

replace

```
print(name.replace('e', '555'))
```

### 分割

split

```
print(name.split(','))
```

## 列表

定义：处理有序项目的数据结构

格式 列表名 = [元素1，元素2......]

## 操作

添加元素

append， extend， insert

```
Arrs = [1,2,3]
Arrs.append(6)  //全部添加 
extend // 字符串分散添加
Arrs.insert(1,6) 指定位置添加
print(Arrs)
```

修改元素

通过下标修改

查找元素

in ， not in ， index， conut

```
Arrs = [1,2,3]

print(1 in Arrs)
print(Arrs.index(1) )
```

删除元素

del， pop， remove

del 列表名

del 列表名[下标]

排序

sort    reverse
